# 기여 안내 (Contributing)

이 프로젝트는 여러분의 기여와 제안을 환영합니다. 대부분의 기여를 위해서는 여러분이 기여물을 사용할 수 있는 권리를 저희에게 부여할 권한이 있으며 실제로 부여한다는 것을 선언하는 기여 라이선스 동의서(CLA)에 동의해야 합니다. 자세한 내용은 https://cla.microsoft.com을 방문하세요.

풀 리퀘스트(PR)를 제출하면 CLA 봇이 자동으로 CLA 제공 필요 여부를 판단하고 PR에 적절한 표시(예: 라벨, 댓글)를 남깁니다. 봇이 제공하는 안내를 따르기만 하면 됩니다. 저희의 CLA를 사용하는 모든 저장소에서 이 작업은 한 번만 수행하면 됩니다.

이 프로젝트는 [Microsoft 오픈 소스 행동 강령](https://opensource.microsoft.com/codeofconduct/)을 채택하였습니다. 자세한 정보는 [행동 강령 FAQ](https://opensource.microsoft.com/codeofconduct/faq/)를 참조하시거나 추가 질문이나 의견이 있으시면 [opencode@microsoft.com](mailto:opencode@microsoft.com)으로 문의해 주시기 바랍니다.
